# Recruitment Review Bot
Telegram-бот для сбора анонимных отзывов о процессе найма в IT-компаниях.


## Tech Stack
**Backend:** Python 3.11+, FastAPI, SQLAlchemy 2.0, PostgreSQL 15+
**Bot:** aiogram 3.x
**Frontend:** React 18, TypeScript, Vite, TanStack Query, Tailwind CSS
**Testing:** pytest, Playwright, Testcontainers

## Project Structure
\\\
recruitment-bot/
├── backend/    # FastAPI REST API
├── bot/        # Telegram bot
├── frontend/   # React admin panel
└── .github/    # CI/CD workflows
